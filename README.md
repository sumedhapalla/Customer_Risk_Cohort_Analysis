📊 Customer Risk & Financial Analytics Dashboard
🚀 Project Overview
This project combines Power BI dashboards and financial data analysis to uncover insights on customer churn behavior and market volatility trends.

It demonstrates skills in data visualization, analytics, and financial modeling.
